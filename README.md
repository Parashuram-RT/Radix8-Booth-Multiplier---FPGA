# Radix8-Booth-Multiplier---FPGA
32-bit Radix-8 Booth Multiplier designed in Verilog HDL and implemented on DE10 Cyclone V FPGA. Features Booth recoding, partial product generation, FSM-based control, ModelSim verification, Quartus synthesis, timing analysis, SignalTap hardware validation, and efficient FPGA resource utilization.
# Radix-8 Booth Multiplier FPGA

## Overview
This project implements a 32-bit signed Radix-8 Booth Multiplier using Verilog HDL and targets the DE10 Cyclone V FPGA platform.

## Features
- Radix-8 Booth Encoding
- Signed 32-bit Multiplication
- FSM-Based Control Logic
- ModelSim Verification
- Quartus Prime Synthesis
- SignalTap Hardware Validation
- FPGA Implementation on DE10 Cyclone V

## Files
- radix8_booth_multiplier.v : Main Verilog Design
- tb_radix8_booth_multiplier.v : Testbench

## Tools Used
- Verilog HDL
- ModelSim
- Intel Quartus Prime
- DE10 Cyclone V FPGA
